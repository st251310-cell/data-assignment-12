# データ分析入門課題:学習時間とテスト得点の関係を調べよう

## 概要

あるクラス40人の「1週間の学習時間」「平均睡眠時間」「テストの得点」を記録した架空のデータ(`data/score.csv`)を分析し、データから何が言えるかを考える課題です。

## 提出までの流れ

1. このリポジトリのページで **「Use this template」→「Create a new repository」** を押し、自分のアカウントにリポジトリを作成する(リポジトリ名は `data-assignment-出席番号` とする)
2. 作成したリポジトリの **Settings → Collaborators → Add people** から、**先生(`GreatT0511`)をコラボレーターとして招待**する
3. 下の課題に取り組み、このREADME.mdの解答欄を書き換える
4. 作成したグラフ(png形式)を `output/` フォルダに保存する
5. 変更を**コミット**する(**最低3回**、意味のある単位でコミットすること)
6. 自分のリポジトリのURLを提出する

> **注意:** テンプレートから作ったリポジトリはフォークと違い、元のリポジトリとつながっていない独立したコピーです。先生を招待しないと提出物を確認できないので、**手順2を忘れずに**。

> コミットメッセージの例:「課題1の代表値を記入」「散布図を追加」のように、何をしたか分かるように書くこと。

## データの説明

`data/score.csv` には次の4列が含まれます。

| 列名 | 説明 |
|---|---|
| 出席番号 | 生徒の番号(1〜40) |
| 学習時間 | 1週間あたりの学習時間(時間) |
| 睡眠時間 | 1日あたりの平均睡眠時間(時間) |
| テスト得点 | 100点満点のテストの得点 |

分析には表計算ソフト(Excel / Googleスプレッドシート)またはPythonのどちらを使ってもかまいません。
